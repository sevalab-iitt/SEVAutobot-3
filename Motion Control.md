# Wheel Alignment - 
       Mecanum wheel is an omnidirectional wheel design (can move in any direction)
       Wheel must be installed in X configuration (frot left front right rear left rear right)
       Each Mecanum wheel is connected with one DC geared motor - these motors provide torque required for the omnidirectional movements
###### Brushed DC motors , this motor provides rotational motion, while the gear box reduces the output speed and increases the torque
